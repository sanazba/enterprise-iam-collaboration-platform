# enterprise-iam-collaboration-platform
### IAM & Collaboration Platform Automation
Simple, production-ready automation for Okta, Google Workspace, and Slack using n8n.
What This Does
Automates common IAM tasks:

✅ Employee Onboarding: Create accounts in Okta, Google, and Slack
✅ Employee Offboarding: Suspend accounts and backup data
✅ Inactive User Detection: Find and suspend inactive accounts
✅ MFA Enforcement: Make sure everyone has MFA enabled

Tech Stack

n8n: Workflow automation
Kubernetes: Container orchestration
PostgreSQL: Store workflow data
Python: Helper scripts
